# Backup
Very basic scripts to backup and restore my home

## Usage

### Edit what you want backup / restore file

Edit `src_dirs` variable at begin of `backup.sh` file

### Backup

```shell
$ ./scripts/backup.sh
```

### Restore

```shell
$ ./scripts/restore.sh /full/path/to/archive.tgz
```
